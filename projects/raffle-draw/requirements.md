# Lottery API

### Requirements

- sell new ticket
- sell multiple tickets at once to a single user
- each ticket will have the following properties
  - ticket id
  - username
  - price
  - cratedAt
  - updatedAt
- get all tickets
- get ticket by id
- get tickets by username
- update ticket by id
- update tickets by username
- delete ticket by id
- delete tickets by username
- draw to get winners
